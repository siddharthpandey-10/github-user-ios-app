# 📱 GitHub User iOS App

A clean and modern iOS app built using **SwiftUI** that allows users to search GitHub profiles and view repositories.

---

## 🚀 Features

* 🔍 Search GitHub users
* 👤 View profile details
* 📦 View repositories
* ⚡ Fast and responsive UI
* 🎨 Modern SwiftUI design

---

## 🛠 Tech Stack

* SwiftUI
* MVVM Architecture
* URLSession (API calls)
* GitHub REST API
